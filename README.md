 Country Information System

 Overview

The Country Information System is a Python-based application that retrieves real-time country information using the REST Countries API. The system allows users to search for individual countries, generate reports for multiple countries, maintain a search history, and analyze country data through various statistics and filtering options.

The project demonstrates the use of API integration, Excel automation, file handling, exception handling, and basic Data Structures and Algorithms (DSA) concepts.

 Features

* Search for a country and view details such as:

  * Country Name
  * Capital
  * Population
  * Region
  * Currency
  * Timezone

* Generate reports for multiple countries entered by the user.

* Save country information and search history into Excel files using openpyxl.

* Create summary reports containing:

  * Total Countries
  * Highest Population Country
  * Lowest Population Country
  * Total Regions Covered

* Search countries using country codes.

* Search countries by capital city.

* Calculate average population of saved countries.

* Prevent duplicate country entries.

* Maintain and view search history.

* Clear search history when required.

 Learning Outcomes

This project provides practical experience in working with APIs, handling real-world data, generating Excel reports, and implementing basic DSA concepts using Python. It serves as a beginner-friendly project for understanding how external data sources can be integrated into Python applications and stored for analysis.
